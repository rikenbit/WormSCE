# WormSCE
SingleCellExperiment objects of C. elegans

Installation
======
```bash
git clone https://github.com/rikenbit/WormSCE/
cd WormSCE
snakemake -j 1
```

# Prerequisites
- Snakemake (5.20.0 or higher)
